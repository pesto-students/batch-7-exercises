## Instructions


Q1.
Creates a new list out of the two supplied by creating each possible pair
from the lists.

### Important 
- Don't use `for-loops`.
- Use only **functional** array methods.

```js
xprod([1, 2], ['a', 'b']); //=> [[1, 'a'], [1, 'b'], [2, 'a'], [2, 'b']]
xprod([a, b], [c, d]) = [[a, c], [a, d], [b, c], [b, d]]
```
